# ITU2019-project

Webová aplikace rezervace místností

Aplikace se nachází zde:
https://duke-of-karamel.github.io/ITU2019-project/

Serverová část aplikace běží na této adrese:
https://vevesoft.net/ituproject

Klientská část běží na:
https://duke-of-karamel.github.io/ITU2019-project/

Zprovoznění jinde:

Server vyžaduje php a MySql.
MySql databáze jde založit pomocí sql query exportované v backend/db_structure.txt
Údaje k databázi se nastavují v souboru backend/db_connect.php

Klientskou část stačí umístit kamkoliv a přes soubor index.html se aplikace spustí.
Pro spuštění backendu na jiném serveru je třeba zkopírovat na server celý obsah složky backend.
Ve frontendu je třeba změnit URL adresy na nový server.
